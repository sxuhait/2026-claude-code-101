### 4. 공유거리
- **신기했던 것:**
  - "index.html을 브라우저에서 열어서 확인할 수 있게 해줘"라고만 말했는데, Claude Code가 직접 서버를 띄워서 바로 결과 화면을 볼 수 있게 해준 점이 신기했습니다. 실행 환경을 직접 세팅할 필요 없이 바로 확인이 가능했습니다.

  - Windows에서는 안내받은 Ctrl+Esc 단축키가 시스템 시작 메뉴 단축키와 겹쳐서 Claude Code 패널이 아니라 Windows 시작 메뉴가 뜨는 문제가 있었습니다. (터미널에 claude 입력하는 방식으로 대신 해결)
