# Members

참여자별 학습 기록·실습·프로젝트를 이 폴더에 정리합니다.

## 폴더 만들기
1. `members/<본인-github-id>/` 폴더를 생성합니다.
2. 주차별 정리는 `week01.md` ~ `week06.md`로 남깁니다.
3. 실전 프로젝트는 `project/` 하위 또는 별도 레포 링크로 관리합니다.

## 예시 구조
```
members/
└── seungwee-choi/
    ├── week01.md
    ├── week02.md
    ├── project/          # 3주차~ 실전 프로젝트
    └── retrospective.md  # 6주차 회고
```
