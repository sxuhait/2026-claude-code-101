## 🧰 주요 개념 요약
| 기능 | 한 줄 설명 |
| --- | --- |
| **CLAUDE.md** | 프로젝트 컨텍스트/규칙을 Claude에게 상시 주입 |
| **Hooks** | 특정 이벤트(툴 실행 전후 등)에 스크립트 자동 실행 |
| **Slash Commands** | 자주 쓰는 프롬프트/워크플로우를 명령어로 등록 |
| **Skills** | 특정 작업용 지침 묶음을 재사용 가능하게 패키징 |
| **MCP** | 외부 도구·데이터 소스를 표준 프로토콜로 연결 |
| **Agent SDK** | Claude Code 기반 커스텀 에이전트 개발 |
