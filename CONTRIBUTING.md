# 참여 가이드 (CONTRIBUTING)

이 스터디는 **직접 손을 움직이며** Claude Code를 익히는 것을 목표로 합니다.
아래 규칙을 따라 학습 기록과 프로젝트를 공유해 주세요.

## 1. 폴더 규칙

- 주차별 공통 자료: `weekNN/` (멘토·전체 공유용)
- 개인 학습 기록: `members/<github-id>/`
  - 예) `members/seungwee-choi/week01.md`
- 실전 프로젝트(3주차~): `members/<github-id>/project/` 또는 별도 레포 링크

## 2. 브랜치 & PR

- 작업은 `main`에서 직접 하지 말고 브랜치를 파세요.
  - 브랜치 이름 예: `week01/<github-id>`, `feat/<github-id>-project`
- 학습/실습 내용을 커밋한 뒤 **Pull Request**를 올립니다.
- PR 제목 예시: `[week01] <github-id> 설치 & 첫 실습 정리`

## 3. 커밋 메시지

- 한글/영어 무관, **무엇을 했는지** 알 수 있게 작성합니다.
- 예: `week02: CLAUDE.md 작성 실습 정리`

## 4. 매주 체크인

- 각 주차 폴더의 체크리스트를 채우고, 막힌 부분은 **Issue**로 남겨 주세요.
- 라벨: `question`, `blocker`, `share` 등을 활용합니다.

## 5. 도움 요청

막히면 혼자 붙잡지 말고 공유하세요. 함께 푸는 게 이 스터디의 핵심입니다. 🙌
